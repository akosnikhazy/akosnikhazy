Untitled generic logic game

I am developing a game from scratch for Steam release. It will be a functional game that I do not want to be big or famous, my prediction is that it will not even make back the Steam release fee.

The only reason I am making this game is because I like the idea of making games as a hobby and I want to make a project from the beginning to release instead of small projects that go nowhere.

I use Unity 3D and C# to develop the game, and Blender for the assets. Every problem I face is a learning experience. I blog about its development on LinkedIn of all places.

Open source, single user financial web application

Since I was a child, I have always documented my money. I started in a notebook. Then I used a very complicated Google Sheet for many years. Now I use GNU Cash to manage my finances like a business.

I like double-entry bookkeeping, but GNU Cash is way too offline and I have to enter my money data at the end of the day instead. I have tried other applications, but everything is way too complicated, and you need trust to keep this information online.

So I decided to develop my own simple application. I will host it on a VPS for myself. So I can use it and enter my data on the go. It will also be free and open source for others who want something like this in their lives.

Some things I made and still like to have around

My hobby of creating random things online generates things like the Color Sudoku game where you solve puzzles without numbers. Speaking of numbers: My Cistercian numeral generator will give you hours of medieval fun! A more serious thing is the self contained encrypter. It can generate an HTML file that asks for a password to decrypt the text or even images you put in it. It also works offline.

Once I even made a tool to visualise temperature data. It creates an image where each row is a year and each column is a day. I am just happy with how it turned out.
